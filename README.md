# Linux Text Editing & File Permissions Exercise

## Overview
This project covers basic **Linux text editing** using **nano** and **vi/vim** editors, and **file permissions management** with `chmod`. It helps develop skills for editing files and controlling access in Linux environments.

## Objective
- **Text Editing**: Learn to create and edit files with **nano** and **vi/vim**.
- **File Permissions**: Understand how to manage file permissions using **numeric** and **symbolic** modes with `chmod`.
